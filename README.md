## About Project

this project develop using laravel framework. this project is only for education & practice purpose.

## Run Project

```bash
# download all needed package from composer
composer install

# download all needed package from npm
npm install

# make .env file from .env.example
cp .env .env.example

# generate new key
php artisan key:generate

# run additional dev pakage from npm (run on separated terminal)
npm run dev

# run project
php artisan serve
```

# Build Project

```bash
# build production ready css and js using npm
npm run build
```
